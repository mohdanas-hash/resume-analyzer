# resume-analyzer
The Resume Analyzer is a web-based platform designed to help students and professionals evaluate and improve their resumes with ease. Built with a focus on clarity and efficiency, it scans resumes to highlight strengths, detect weaknesses, and provide actionable suggestions for improvement.

🔹 Key Features:

ATS Compatibility Check – Ensures resumes are optimized for Applicant Tracking Systems.

Keyword Analysis – Identifies missing industry-specific terms to boost visibility.

Formatting Review – Suggests improvements in layout, structure, and readability.

Skill & Experience Match – Compares resume content with job descriptions for better alignment.

Instant Feedback – Provides concise, exam-ready style recommendations for quick edits.

# Resume Analyzer 📄🔍

A rule-based resume inspection tool built with HTML, CSS, and JavaScript. 

## ✨ Features
* **Format Check:** Scans for standard sections like Education, Experience, and Skills.
* **Phrase Analysis:** Flags weak action verbs and overused cliches.
* **Keyword Matching:** Paste a target job description to check your keyword overlap percentage.
* **Dark Mode:** Built-in toggle for comfortable viewing.
